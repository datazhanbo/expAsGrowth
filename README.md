 
# 实验即增长：AI 时代的增长方法论

> 增长不是一次猜对，而是持续验证；不是单点爆发，而是实验复利。

## 关于本书

AI 时代，增长的逻辑并没有变化。无论技术如何迭代，从因果推断到实验验证的基本方法论，仍是产品增长、商业化增长和买量投放中最主要的方法论。本书系统阐述「实验即增长」的核心理念，结合内容平台、电商、出海游戏等行业的实战经验，提供从方法论到平台建设的完整路径。

## 核心观点

1. **实验是增长的底层逻辑** —— 增长不是靠灵感，而是靠系统化的假设-验证-迭代循环
2. **AI 时代没有改变增长的本质** —— 模型是工具，实验是方法，因果推断仍是决策的基础
3. **产品、商业化、买量是增长的三元闭环** —— 任何一环的优化都依赖实验，任何一环的短板都会拖垮全局
4. **平台建设的目标是决策系统** —— 不是功能堆砌，而是让实验结论可积累、可复用、可自动化
